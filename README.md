# Library Management System

This project is a desktop application developed in C# for managing a library system.
The application allows users to manage books, users, and subscriptions through a graphical interface. It supports multiple user roles (admin, librarian, user), each with different access levels and functionalities.
The system includes authentication features for user login and account management, and it is connected to a SQL database for storing and retrieving data. Basic cryptography methods are used to handle sensitive information securely.
Automated tests are implemented to ensure that core functionalities work correctly and reliably.

## Technologies
- C#
- .NET (Windows Forms)
- SQL Database

## Features
- User authentication and login system
- Role-based access (admin, librarian, user)
- Management of books, users, and subscriptions
- Database integration for data storage
- Graphical user interface (Windows Forms)
- Basic cryptography for sensitive data
- Automated tests for functionality validation
